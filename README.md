# 💱 Convert - Conversor de Moedas

Projeto desenvolvido durante os estudos na plataforma **Rocketseat**, com o objetivo de praticar **HTML, CSS e JavaScript** criando um conversor simples de moedas para Real Brasileiro (BRL).

O usuário informa um valor, seleciona a moeda estrangeira e o sistema calcula automaticamente o valor equivalente em reais.

---

## 🚀 Demonstração

Interface do projeto:

- Input para inserir o valor
- Seleção de moeda estrangeira
- Botão para converter
- Exibição da cotação e do resultado da conversão

---

## 🧠 Funcionalidades

✔ Inserir valor para conversão  
✔ Selecionar moeda estrangeira  
✔ Converter automaticamente para **Real Brasileiro (BRL)**  
✔ Exibir a cotação da moeda selecionada  
✔ Mostrar o resultado formatado em reais  
✔ Validação para impedir valores inválidos  
✔ Interface moderna e responsiva  

---

## 💰 Moedas disponíveis

## 🔄 Como funciona a conversão

1. O usuário digita o valor no campo **Valor**
2. Seleciona a moeda estrangeira
3. Clica em **Converter em reais**
4. O JavaScript calcula:

valor digitado × cotação da moeda


5. O resultado é exibido na tela formatado em **R$**

Exemplo:
US$ 1 = R$ 4,87

100 USD = 487,00 Reais
